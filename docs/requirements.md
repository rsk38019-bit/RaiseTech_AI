## 1. プロジェクト概要

### 1.1 目的

シンプルなカンバンボード形式のタスク管理アプリを提供する。ユーザーがタスクをカード単位で管理し、進捗状況をカラム間の移動で可視化できるようにする。

### 1.2 スコープ

- シングルページアプリケーション（SPA）
- フロントエンドから REST API 経由でバックエンドと通信する
- データは PostgreSQL に永続化する
- AWS 上にデプロイしインターネットからアクセスできる

### 1.3 対象外（スコープ外）

- ユーザー認証・アカウント管理
- 複数ボードの管理
- カード詳細（説明文・期日・担当者・ラベル）
- ドラッグ&ドロップによるカード移動
- チームコラボレーション機能
